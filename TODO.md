## 📌 ЗАМЕТКИ ПО ML

- **Автоповорот** может требовать удаления фона для стабильной работы (алгоритм ищет контрастные края)
- **Уменьшение при повороте** — следствие отсутствия повторного кадрирования
- **Центрирование** работает, требует оптимизации для разных форм значков
- **Поиск похожих** работает на основе цветовых гистограмм (512) + ORB (32). Для работы нужно пересчитать векторы через админ-панель.
